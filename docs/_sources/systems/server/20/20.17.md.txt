
### 20.17  
Configure an application allow-listing program to employ a deny-all, permit-by-exception 
policy to allow the execution of authorized software programs. 
If AppLocker is used, it is configured through group policy: 
Computer Configuration\Windows Settings\Security Settings\Application Control 
Policies\AppLocker 
Implementation guidance for AppLocker is available in the NSA paper: Application 
Whitelisting using Microsoft AppLocker.
