
### 20.40  
Remove any unauthorized or standard user accounts from the Administrators group. 
Stand-alone system 
  Open Local Users and Groups 
  Navigate to Groups 
  Review the Administrators group for unauthorized accounts or standard user 
accounts that should not have administrator privileges. 
Remove any unauthorized or standard user accounts. 
Domain-joined system 
  Open Active Directory Users and Computers 
  Review the Administrators and Domain Admins groups (which must be replaced 
with a domain member server administrator group) group for unauthorized 
accounts or standard user accounts that should not have administrator privileges. 
Remove any unauthorized or standard user accounts.
