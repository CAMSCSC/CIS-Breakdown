To establish the recommended configuration via GP, set the following UI path to: `Not Installed`. `Computer Configuration\Policies\Windows Settings\Security Settings\System Services\Microsoft FTP Service `OR To Uninstall the FTP Server role:  Start Server Manager  Select the server with the role  Scroll down to ROLES AND FEATURES in the right pane  Select Remove Roles and Features from the drop-down TASKS list  Select the appropriate server on the Server Selection page and click Next  Deselect FTP Server under Web Server (IIS) on the Roles page  Click Next and Remove as prompted (if installed).
